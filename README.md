## About

MS Computer Science student at Georgia Tech (2025–2027), currently working as an AI Trainer at Outlier and as a Research Technician at the University of Winnipeg. Previously a Business Technology Analyst at Hang Seng Bank. Looking for full-time roles in AI/ML and data engineering starting Summer 2025.

## Projects

- **Autonomous Robotics** — ROS2-based navigation system built out of agricultural robotics research; implements 30 Hz EKF sensor fusion across wheel odometry, IMU, and GPS with Ackermann steering constraints and a thread-safe emergency stop architecture.
- **Shoe Brand Classifier** — VGG16 transfer learning model trained on 15K+ images, achieving 82% validation accuracy; built for automated brand detection in social media marketing workflows.
- **Customer Segmentation Pipeline** — K-means clustering on mall customer data identifying 4 market segments; includes automated preprocessing, elbow/silhouette cluster selection, and exportable reports.

## Stack

React 18, TypeScript, Tailwind CSS, Vite, GitHub Pages, GitHub Actions

## Local setup

```bash
git clone https://github.com/ttkw-git/ttkw-git.github.io.git
cd ttkw-git.github.io
npm install
npm run dev       # dev server
npm run build     # production build
npm run preview   # preview production build
```

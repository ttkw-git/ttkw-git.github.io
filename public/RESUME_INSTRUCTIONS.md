# Resume PDF Instructions

## Where to Place Your Resume

Place your resume PDF file in this directory (`public/`) with the following name:

```
public/Tony_Wong_Resume.pdf
```

## How to Create Your Resume PDF

1. Create or update your resume document (Word, Google Docs, etc.)
2. Export/Save as PDF
3. Name it: `Tony_Wong_Resume.pdf`
4. Place it in this `public` folder

## Important Notes

- The file name MUST be exactly: `Tony_Wong_Resume.pdf`
- The resume link is already configured in the website to point to: `/Tony_Wong_Resume.pdf`
- Make sure the file size is reasonable (< 2MB recommended)
- Test the download link after placing the file
